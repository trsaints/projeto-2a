# projeto-2a

O projeto "Agendai" é um sistema de agenda para visualização de calendário, agendamento de eventos, tarefas e controle de turnos.
Esta aplicação foi desenvolvida para a disciplina de Projeto Integrador 2A, do curso de Ciência da Computação na instituição de ensino IESB

## Dependências

Este projeto utiliza as seguintes bibliotecas para seu funcionamento:

- [AvaloniaUI]()
- [EntityFrameworkCore]()
- [SQLite]()
- [AvaloniaMaterialDesign]()

## Requisitos

### 1. Calendário

- Visualização de calendário
- Agendamento de eventos
- Agendamento de tarefas

### 2. Lista de tarefas

- Visualização de tarefas
- Agendamento de tarefas
- Histórico de tarefas

### 3. Turnos

- Controle de turnos com pomodoro
- Histórico de turnos
- Agendamento de turnos
